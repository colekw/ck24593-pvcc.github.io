Term Definition
LAN Local Area Network - A network that connects computers and devices within a limited geographic area, such as a home, office, or campus.
MAN Metropolitan Area Network - A network that covers a larger geographic area than a LAN but is smaller than a WAN, typically serving a city or a large campus.
WAN Wide Area Network - A network that spans a large geographic area, connecting multiple LANs and MANs, often over long distances.
Network models Conceptual frameworks that describe the structure and functions of computer networks, such as the OSI and TCP/IP models.
Bus network A network topology where all devices share a single communication line, or bus, enabling data to be transmitted to all devices on the network.
Star network A network topology where all devices are connected to a central hub or switch, facilitating efficient communication between devices.
Protocols Set of rules that govern how data is transmitted over a network, including CSMA/CD Ethernet and CSMA/CA WiFi.
Combining networks The integration of multiple networks using devices such as repeaters, bridges, switches, routers, and gateways.
Internet Addressing The process of assigning unique identifiers, such as IP addresses, to devices on the Internet for communication.
Tier 2 Network infrastructure providers that connect to Tier 1 providers and smaller ISPs to improve Internet accessibility.
Tier 1 The highest level of Internet service providers that have direct connections to the global Internet backbone.
IP address A numerical label assigned to each device participating in a computer network that uses the Internet Protocol for communication.
ICANN Internet Corporation for Assigned Names and Numbers - A non-profit organization responsible for coordinating the global Internet's systems of unique identifiers.
DNS Domain Name System - A hierarchical system that translates human-readable domain names into IP addresses for routing on the Internet.
TLD: Top-level domain The highest level in the hierarchical Domain Name System, such as .com, .org, and .net.
Country-code TLDS Top-level domains associated with specific countries or territories, such as .us for the United States and .uk for the United Kingdom.
Internet protocols Standardized rules that define communication on the Internet, including protocols like FTP, HTTP, and SMTP.
World Wide Web A system of interlinked hypertext documents accessed via the Internet using browsers, with key components including URLs, client-side activities, and server-side activities.
URL Uniform Resource Locator - A web address that specifies the location of a resource on the Internet.
Client-side activities Processes and actions performed on the user's device within a web application or website.
Server-side activities Processes and actions performed on the web server in response to requests from client-side activities.
Internet four-layer software The layered architecture of Internet software, consisting of the application layer, transport layer, network layer, and link layer.
Application layer The top layer of the Internet software architecture responsible for providing network services directly to end-users.
Transport layer The layer in the Internet software architecture responsible for end-to-end communication and data flow control.
Network layer The layer in the Internet software architecture responsible for routing and logical addressing.
Link layer The lowest layer in the Internet software architecture responsible for data link and physical layer functions.
Internet A global network of interconnected computers and devices using standardized communication protocols.
Packet-switching network A network in which data is broken into packets for efficient transmission over various routes and reassembled at the destination.
Fault-tolerant The ability of a system or network to continue functioning in the presence of hardware or software failures.
Redundant Having duplicate components or systems in place to ensure continued operation in case
